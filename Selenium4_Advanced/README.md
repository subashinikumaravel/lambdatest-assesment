# Selenium Advanced Assignment - LambdaTest

## Project structure
- `src/test/java/...` : Test classes and BaseTest
- `testng.xml` : TestNG suite that runs two tests in parallel (class-level)
- `pom.xml` : Maven configuration

## Pre-requisites
- Java 17+ and Maven installed (or use Gitpod)
- LambdaTest account
- Set the following environment variables:
  - `LT_USERNAME` : your LambdaTest username
  - `LT_ACCESS_KEY` : your LambdaTest access key

## How to run locally
1. Set environment variables (example on Unix/macOS):
   ```bash
   export LT_USERNAME=your_username
   export LT_ACCESS_KEY=your_access_key
